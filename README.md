# dotfiles

hyprland, waybar, nvim, alacritty, rofi, and other linux configs.

managed with [gnu stow](https://www.gnu.org/software/stow/).

## setup

```bash
git clone https://github.com/ax1g/dotfiles ~/dotfiles
cd ~/dotfiles
./setup.sh
```

## what's inside

| package | links to |

| hypr | ~/.config/hypr/ |
| waybar | ~/.config/waybar/ |
| nvim | ~/.config/nvim/ |
